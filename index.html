TextInNotepad (TIN)


What is TIN?

A python based computer malware that writes text into notepad.

What does TIN do?

TIN locks down the user's keyboard and forces the mouse position to (0, 0). Then it copies itself to the startup folder, followed by opening notepad and writing text in. TIN also has an anti Ctrl+Alt+Del system. TIN constantly checks if Task Manager is open and if it is opened, TIN will instantly shut the computer down.
